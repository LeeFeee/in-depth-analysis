# In-depth Analysis / 深度分析

> 一个通用 AI 深度分析技能，兼容 Claude Code、OpenClaw、Hermes Agent 等所有支持 Skill 的 AI 产品，基于「深度分析法」生成完整的深度分析报告（In-depth Analysis Report）。

## 功能

通过两个维度对任意研究对象进行深度分析，产出结构化的深度分析报告：

- **历史脉络分析** — 沿时间轴还原研究对象从诞生到现在的发展全貌
- **全景生态分析** — 以当前时间点为切面，与同赛道竞品/同类进行全面对比
- **综合研判** — 结合历史脉络与生态格局，给出综合判断

适用于产品/工具、公司/组织、技术概念、人物等任何研究对象的完整深度分析。

## 安装

在对话中告诉 AI：

```
安装 https://github.com/LeeFeee/in-depth-analysis 这个 skill
```

支持 Claude Code、OpenClaw、Hermes Agent 等所有兼容 Skill 的 AI 产品。

## 使用

使用以下触发词：

```
"深度分析 XXX"
"研究一下 XXX"
"分析一下 XXX 的前世今生"
"做个深度研究报告"
"写一份行业研究"
```

## 报告结构

```markdown
# 深度分析法：{研究对象}

## 一、历史脉络分析    （6,000-15,000 字）
## 二、全景生态分析    （3,000-10,000 字）
## 三、综合研判        （1,500-3,000 字）
```

全文建议 10,000-30,000 字。
